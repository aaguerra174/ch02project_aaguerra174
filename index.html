Building HTML Lists

  1. embedded style sheet: Styling information written in the top portion of an HTML page.
  2. external style sheet: A seperate file containing CSS rules to control the presentation and layout of HTML pages.
  3. inline style: Styling information for an HTML tag that is written inside of, that is, inline with, the HTML tag as a set of attributes.
